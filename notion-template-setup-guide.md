# 🛠️ Notion 模板快速设置指南

> 5 分钟快速搭建你的开发者工作流

---

## 📋 准备工作

1. 确保你有 Notion 账号（免费版即可）
2. 创建一个新的 Notion 页面作为工作区
3. 准备好你的技术栈列表（用于标签设置）

---

## 🚀 快速设置步骤

### 步骤 1: 创建主页面结构

在 Notion 中创建以下页面结构：

```
🚀 开发者工作流
├── 📁 项目管理
├── 📚 学习路径
├── 💻 代码片段库
├── 📝 技术笔记
├── ✅ 任务管理
└── 🔖 资源收藏
```

**操作：**
- 输入 `/page` 创建新页面
- 使用 `/icon` 添加图标
- 使用 `/cover` 添加封面

---

### 步骤 2: 创建项目管理数据库

1. 在「📁 项目管理」页面输入 `/table` 创建数据库
2. 重命名数据库为「项目管理」
3. 添加以下属性：

| 属性名 | 类型 | 选项/说明 |
|--------|------|-----------|
| 项目名称 | Title | (默认) |
| 状态 | Select | 🟢 进行中, 🟡 规划中, 🔵 已完成, ⚫ 已暂停 |
| 优先级 | Select | 🔴 高, 🟠 中, 🟢 低 |
| 技术栈 | Multi-select | React, Vue, Node.js, Python, TypeScript, Go, Java... |
| 开始日期 | Date | |
| 截止日期 | Date | |
| 进度 | Number | 格式：百分比 (0-100) |
| 项目链接 | URL | GitHub/GitLab 链接 |
| 项目描述 | Text | |

4. 创建视图：
   - **看板视图**：按「状态」分组
   - **日历视图**：按「截止日期」
   - **优先级视图**：按「优先级」排序

---

### 步骤 3: 创建学习路径数据库

1. 在「📚 学习路径」页面创建数据库
2. 添加属性：

| 属性名 | 类型 | 选项/说明 |
|--------|------|-----------|
| 课程/资源名称 | Title | |
| 类型 | Select | 📖 教程, 🎥 视频, 📝 文档, 🛠️ 实战项目 |
| 难度 | Select | ⭐ 入门, ⭐⭐ 中级, ⭐⭐⭐ 高级 |
| 状态 | Select | 📌 待学习, 📖 学习中, ✅ 已完成 |
| 标签 | Multi-select | Frontend, Backend, DevOps, Mobile, AI... |
| 学习日期 | Date | |
| 完成日期 | Date | |
| 资源链接 | URL | |
| 评分 | Number | 1-5 |

3. 创建视图：
   - 按「状态」分组
   - 按「难度」筛选
   - 按「标签」分类

---

### 步骤 4: 创建代码片段库

1. 在「💻 代码片段库」页面创建数据库
2. 添加属性：

| 属性名 | 类型 | 选项/说明 |
|--------|------|-----------|
| 片段名称 | Title | 如 "React Hook: useDebounce" |
| 语言 | Select | JavaScript, TypeScript, Python, Go, Java, C++... |
| 分类 | Select | 🎣 Hooks, 🛠️ Utils, 🎨 Components, 🔧 Config |
| 标签 | Multi-select | React, Performance, API, State... |
| 代码 | Code | 代码块内容 |
| 使用场景 | Text | |
| 示例 | Text | |
| 创建日期 | Date | |
| 使用次数 | Number | |

3. 创建视图：
   - 按「语言」分组
   - 按「分类」筛选
   - 「最常用」：按「使用次数」降序

---

### 步骤 5: 创建技术笔记数据库

1. 在「📝 技术笔记」页面创建数据库
2. 添加属性：

| 属性名 | 类型 | 选项/说明 |
|--------|------|-----------|
| 笔记标题 | Title | |
| 分类 | Select | 📖 概念, 🐛 Bug 解决, 💡 最佳实践, 🔍 源码分析 |
| 标签 | Multi-select | React, Vue, Node.js, Algorithm... |
| 创建日期 | Date | |
| 更新日期 | Date | |
| 关联项目 | Relation | 关联到「项目管理」数据库 |
| 关联学习 | Relation | 关联到「学习路径」数据库 |

3. 创建页面模板：
   - 点击数据库右上角「...」→「Templates」→「New template」
   - 添加以下内容块：

```
## 📌 核心概念

## 💡 关键要点

## 📝 代码示例

## 🔗 相关资源

## ❓ 待解决问题
```

3. 创建看板视图：按「状态」分组

## 🎨 美化设置

### 设置封面

1. 在主页面点击右上角「Add cover」
2. 选择渐变色背景或上传图片
3. 推荐颜色：深蓝到紫色渐变

### 添加图标

1. 在每个页面输入 `/icon`
2. 选择对应的 emoji 图标：
   - 📁 项目管理
   - 📚 学习路径
   - 💻 代码片段库
   - 📝 技术笔记
   - ✅ 任务管理
   - 🔖 资源收藏

### 设置颜色

在数据库属性中，为 Select 类型的选项设置颜色：
- 🟢 进行中/已完成：绿色
- 🟡 规划中：黄色
- 🔵 已完成：蓝色
- 🔴 高优先级：红色

## 💡 使用技巧

### 1. 使用模板按钮

为常用操作创建模板按钮：
- 在页面输入 `/button`
- 设置按钮名称和操作
- 例如：「添加新项目」→ 打开项目管理数据库

### 2. 创建公式属性

在项目管理中添加「剩余天数」公式：
```
dateBetween(prop("截止日期"), now(), "days")
```

### 3. 使用关联关系

充分利用数据库关联：
- 任务关联到项目
- 笔记关联到学习和项目
- 实现知识网络化

### 4. 创建筛选视图

为不同场景创建筛选视图：
- 「本周任务」：筛选本周截止的任务
- 「高优先级项目」：筛选高优先级项目
- 「未读资源」：筛选未读的资源
