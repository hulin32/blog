# ⚡ Notion 模板一键导入 - 快速操作指南

> 3 步创建可一键导入的模板，让用户 5 秒完成设置

---

## 🎯 核心步骤（3 步完成）

### 步骤 1: 创建完整模板
按照 `notion-create-template-step-by-step.md` 创建完整的模板

### 步骤 2: 设置为可分享
1. 打开模板主页面
2. 点击右上角 **"Share"**
3. ✅ 勾选 **"Share to web"**
4. ✅ 勾选 **"Allow duplicate as template"**
5. 复制链接

### 步骤 3: 分享给用户
```
🎉 一键导入模板

点击链接 → 点击 "Duplicate" → 完成！

👉 [模板链接]
```

## 📋 详细操作

### 1️⃣ 创建模板（15-20 分钟）

参考：`notion-create-template-step-by-step.md`

**必须完成：**
- ✅ 创建 6 个数据库
- ✅ 设置所有属性
- ✅ 创建视图
- ✅ 设置关联关系
- ✅ 添加示例数据（推荐）

### 2️⃣ 设置分享权限（1 分钟）

**操作步骤：**

1. **打开模板主页面**
   ```
   进入「🚀 开发者工作流」页面
   ```

2. **点击 "Share" 按钮**
   ```
   页面右上角 → 点击 "Share"
   ```

3. **设置权限**
   ```
   ✅ Share to web（分享到网页）
   ✅ Allow duplicate as template（允许复制为模板）
   权限：Can view（可查看）
   ```

4. **复制链接**
   ```
   点击 "Copy link"
   链接格式：https://www.notion.so/...
   ```

5. **测试链接**
   ```
   在无痕窗口打开
   确认能看到 "Duplicate" 按钮
   ```

### 3️⃣ 用户使用（5 秒）

**给用户的说明：**

```
🚀 一键导入模板

1. 点击下面的链接打开模板
2. 点击右上角的 "Duplicate" 按钮
3. 选择你的工作区
4. 完成！🎉

👉 [点击这里打开模板](你的模板链接)
```

## 🆘 常见问题

**Q: 用户看不到 "Duplicate" 按钮？**
A: 检查是否启用了 "Allow duplicate as template"

**Q: 链接无法访问？**
A: 检查是否启用了 "Share to web"

**Q: 如何更新模板？**
A: 修改原始模板，用户重新复制即可

**Q: 可以设置密码吗？**
A: Notion 不支持密码，但可以设置访问权限
