## 📋 使用流程

### 对于买家：
1. 购买模板
2. 获得模板链接（一键复制）或文档
3. 点击 "Duplicate" 按钮一键导入（推荐）
4. 或按照设置指南手动搭建
5. 开始使用

### 对于卖家：
1. 使用 `notion-template-sales-page.md` 制作销售页面
2. **按照 `notion-template-import-guide.md` 创建可分享的模板链接** ⭐
3. 提供 `notion-create-template-step-by-step.md` 作为详细教程（手动搭建）
4. 提供 `notion-quick-reference.md` 作为快速参考
5. 参考 `notion-template-developer.md` 回答技术问题
