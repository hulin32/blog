# 📖 Notion 模板创建详细教程

> 手把手教你如何在 Notion 中创建开发者工作流模板

---

## 🎯 创建方式说明

在 Notion 中创建模板有两种方式：

### 方式一：手动创建（推荐学习）
- ✅ 完全理解每个功能
- ✅ 可以自定义调整
- ⏱️ 需要 15-20 分钟

### 方式二：复制模板（快速开始）
- ✅ 5 分钟完成
- ✅ 直接使用
- ⚠️ 需要先创建好模板页面

**本教程采用方式一，让你完全掌握创建过程！**

---

## 📋 第一步：创建主工作区

### 1.1 创建新页面

1. 打开 Notion（网页版或 App）
2. 点击左侧边栏的 **"+"** 或 **"New page"**
3. 输入页面标题：**"🚀 开发者工作流"**

### 1.2 添加封面

1. 点击页面顶部的 **"Add cover"** 按钮
2. 选择：
   - **Unsplash** 搜索 "code" 或 "developer"
   - 或者选择渐变色（推荐深蓝到紫色）
   - 或者上传自己的图片

### 1.3 添加图标

1. 点击标题旁边的 **"Add icon"** 按钮
2. 选择 emoji：🚀 或 💻

---

## 📁 第二步：创建子页面结构

### 2.1 创建 6 个子页面

在主页面的空白处，依次输入以下命令创建子页面：

**方法：** 输入 `/page` 然后回车，输入页面名称

创建以下 6 个页面：

1. 输入 `/page` → 输入 **"📁 项目管理"**
2. 输入 `/page` → 输入 **"📚 学习路径"**
3. 输入 `/page` → 输入 **"💻 代码片段库"**
4. 输入 `/page` → 输入 **"📝 技术笔记"**
5. 输入 `/page` → 输入 **"✅ 任务管理"**
6. 输入 `/page` → 输入 **"🔖 资源收藏"**

**提示：** 每个页面创建后，可以点击页面名称旁边的图标按钮，添加对应的 emoji 图标。

## 🗄️ 第三步：创建第一个数据库 - 项目管理

### 3.1 进入项目管理页面

点击刚才创建的 **"📁 项目管理"** 页面

### 3.2 创建数据库

1. 在空白处输入：`/table` 然后回车
2. 选择 **"Table - Inline"**（内联表格）
3. 数据库会自动创建，默认有一个 "Name" 列

### 3.3 重命名数据库

1. 点击数据库左上角的 **"Untitled"**
2. 改为 **"项目管理"**

### 3.4 添加属性（列）

点击数据库右上角的 **"+"** 或点击列标题，添加以下属性：

#### 属性 1: 状态（Select）
1. 点击 **"+"** → 选择 **"Select"**
2. 属性名输入：**"状态"**
3. 点击属性下方的 **"Add options"**，添加以下选项：
   - 🟢 进行中（选择绿色）
   - 🟡 规划中（选择黄色）
   - 🔵 已完成（选择蓝色）
   - ⚫ 已暂停（选择灰色）

#### 属性 2: 优先级（Select）
1. 点击 **"+"** → 选择 **"Select"**
2. 属性名：**"优先级"**
3. 添加选项：
   - 🔴 高（红色）
   - 🟠 中（橙色）
   - 🟢 低（绿色）

#### 属性 3: 技术栈（Multi-select）
1. 点击 **"+"** → 选择 **"Multi-select"**
2. 属性名：**"技术栈"**
3. 添加常用技术栈选项：
   - React
   - Vue
   - Node.js
   - Python
   - TypeScript
   - Go
   - Java
   - Swift
   - Kotlin
   - Flutter
   - （根据你的需要添加更多）

#### 属性 4: 开始日期（Date）
1. 点击 **"+"** → 选择 **"Date"**
2. 属性名：**"开始日期"**

#### 属性 5: 截止日期（Date）
1. 点击 **"+"** → 选择 **"Date"**
2. 属性名：**"截止日期"**

#### 属性 6: 进度（Number）
1. 点击 **"+"** → 选择 **"Number"**
2. 属性名：**"进度"**
3. 点击属性设置 → 格式选择 **"Percent"**（百分比）

#### 属性 7: 项目链接（URL）
1. 点击 **"+"** → 选择 **"URL"**
2. 属性名：**"项目链接"**

#### 属性 8: 项目描述（Text）
1. 点击 **"+"** → 选择 **"Text"**
2. 属性名：**"项目描述"**

### 3.5 创建视图

点击数据库左上角的 **"Table"** 下拉菜单：

#### 视图 1: 看板视图
1. 点击 **"Add a view"** → 选择 **"Board"**
2. 名称：**"看板视图"**
3. 分组方式：选择 **"状态"**
4. 保存

#### 视图 2: 日历视图
1. 点击 **"Add a view"** → 选择 **"Calendar"**
2. 名称：**"日历视图"**
3. 日期属性：选择 **"截止日期"**
4. 保存

#### 视图 3: 优先级视图
1. 点击 **"Add a view"** → 选择 **"Table"**
2. 名称：**"优先级视图"**
3. 点击视图右上角的 **"Sort"** → 按 **"优先级"** 降序
4. 保存

## 📚 第四步：创建学习路径数据库

### 4.1 进入学习路径页面

点击 **"📚 学习路径"** 页面

### 4.2 创建数据库

| 属性名 | 类型 | 选项 |
|--------|------|------|
| 课程/资源名称 | Title | (默认) |
| 类型 | Select | 📖 教程, 🎥 视频, 📝 文档, 🛠️ 实战项目 |

| 状态 | Select | 📌 待学习, 📖 学习中, ✅ 已完成 |
| 标签 | Multi-select | Frontend, Backend, DevOps, Mobile, AI, Data Science... |
| 学习日期 | Date | |
| 完成日期 | Date | |
| 资源链接 | URL | |
| 评分 | Number | 格式：数字（1-5） |

**操作步骤：**
1. 依次点击 **"+"** 添加每个属性
2. 对于 Select 类型，记得添加选项并设置颜色
3. 对于 Multi-select，添加常用标签

### 4.4 创建视图

- **按状态分组**：Board 视图，按 "状态" 分组
- **按难度筛选**：Table 视图，筛选 "难度"

## 📝 第六步：创建技术笔记数据库

### 6.1 进入技术笔记页面

点击 **"📝 技术笔记"** 页面

### 6.2 创建数据库

输入 `/table` → 选择 **"Table - Inline"**

### 6.3 添加属性

| 属性名 | 类型 | 说明 |
|--------|------|------|
| 笔记标题 | Title | (默认) |
| 分类 | Select | 📖 概念, 🐛 Bug 解决, 💡 最佳实践, 🔍 源码分析 |
| 标签 | Multi-select | React, Vue, Node.js, Algorithm, Performance... |

**关联设置：**
1. 添加 "关联项目" 属性时，选择 **"Relation"**
2. 选择关联到 **"项目管理"** 数据库

1. 点击数据库右上角的 **"..."** 菜单
2. 选择 **"Templates"** → **"New template"**
3. 模板名称：**"技术笔记模板"**
4. 在模板页面中添加以下内容：

```
## 📌 核心概念

## 💡 关键要点

## 📝 代码示例

## 🔗 相关资源

## ❓ 待解决问题
```

5. 保存模板

### 6.5 创建视图

- **按日期排序**：Table 视图，按 "创建日期" 降序
- **按标签分类**：Gallery 视图，按 "标签" 分组
- **按分类筛选**：Table 视图，筛选 "分类"

## 💡 高级技巧

### 1. 使用公式计算

在「项目管理」中添加「剩余天数」：

1. 添加新属性：**"剩余天数"**，类型选择 **"Formula"**
2. 输入公式：
```notion
dateBetween(prop("截止日期"), now(), "days")
```

### 2. 创建模板按钮

在主页面：
1. 输入 `/button`
2. 按钮名称：**"添加新项目"**
3. 操作：打开「项目管理」数据库

### 3. 使用筛选视图

创建「本周任务」视图：
1. 在「任务管理」中创建新视图
2. 添加筛选：截止日期 = 本周
3. 保存为「本周任务」

## 🆘 常见问题

**Q: 如何删除不需要的属性？**
A: 点击属性名称 → 选择 "Delete property"

**Q: 如何修改视图？**
A: 点击视图名称 → 选择 "Edit view"

**Q: 如何复制数据库？**
A: 点击数据库右上角 "..." → "Duplicate"

**Q: 代码片段中的代码块在哪里添加？**
A: 点击数据库中的条目，进入页面，然后输入 `/code` 添加代码块
